# 📄 PDF 可视化水印工具使用指南

一款轻量级、可视化的 PDF 批量加水印工具，支持图片和文字水印，提供实时预览与拖拽定位功能。

## 🚀 快速下载
请前往本项目的 [GitHub Releases](https://github.com/inkstar/waterprint_forpdf/releases) 页面下载对应系统的版本：
*   **Windows**: 下载 `PDFWatermark-Windows`，运行 `.exe` 文件。
*   **macOS**: 下载 `PDFWatermark-MacOS.dmg`。

---

## 🍎 macOS 用户安装须知 (重点)

由于本软件未在 Apple Store 上架，首次安装和运行时，macOS 的安全机制（Gatekeeper）可能会拦截。请按照以下步骤操作：

### 1. 安装
双击打开 `.dmg` 文件，将 **PDFWatermark** 图标拖拽到 **Applications (应用程序)** 文件夹中。

### 2. 首次运行权限
如果您双击打开程序时提示 **“无法打开，因为无法验证开发者”** 或 **“来自不明开发者”**：
1.  点击提示框的“取消”。
2.  打开系统的 **“系统设置” (System Settings)** -> **“隐私与安全性” (Privacy & Security)**。
3.  向下滑动找到“安全性”部分，您会看到提示 *“已拦截 PDFWatermark 的使用...”*。
4.  点击旁边的 **“仍要打开” (Open Anyway)**。
5.  在随后的确认框中输入您的开机密码，点击“打开”即可。
    *   *提示：此操作仅需在首次运行时执行一次。*

## 💻 核心功能使用步骤

### 第一步：选择文件
点击 **“选择 PDF”**。支持一次性选择多个 PDF 文件。
*   **预览切换**：如果选择了多个文件，可以使用下方的“上一个/下一个文件”按钮查看不同文档的效果。

### 第二步：添加水印
点击 **“+ 图片水印”** 或 **“+ 文字水印”**。
*   您可以添加多个水印层，它们会显示在“水印列表”中。
*   在列表中选中某个水印后，可以在下方修改其内容、字号、透明度或旋转角度。

### 第三步：位置控制
*   **手动拖拽**：最直观的方式。直接在右侧预览图中用鼠标点击水印并拖动到任意位置。
*   **一键定位**：使用左侧面板的“左上角”、“右上角”或“居中”按钮快速对齐。

### 第四步：输出设置
*   **输出目录**：默认保存在原文件同目录下。点击“选择输出目录”可以将所有文件统一存放到指定文件夹。
*   **文件名后缀**：默认为 `_marked`，您可以自定义（如 `-公司内部用`）。

### 第五步：批量处理
确认预览无误后，点击底部的 **“开始批量处理”**。程序将自动处理列表中的所有文件。
